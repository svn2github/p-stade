A Range Adaptor delivers an altered presentation of one or more underlying ranges. Range Adaptors are lazy, meaning that their elements are only computed on demand. The underlying ranges are not modified, and the traversal category is the same unless otherwise specified. Additional information is available at Range Library Proposal.

adjacent_filtered

Description

adjacent_filtered skips elements while a Predicate applied to adjacent pairs doesn't satisfy.

Header

  • <pstade/oven/adjacent_filtered.hpp>

Model of

  • Pipable Object

Valid expressions

Valid expression Semantics
_fwdrng|adjacent_filtered(_prd) A Forward Range {b0,b1,b2,..,bM}.

Preconditions

  • {a0,a1,a2,a3,..,aN} is _fwdrng.
  • _prd(*_begin(_fwdrng), *_begin(_fwdrng)) is a well-formed.
  • {b0,b1,b2,..,bM} is a subset of {a0,a1,a2,a3,..,aN}.
  • _prd(b0, b1), _prd(b1, b2), .., _prd(bM-1, bM) are true.
  • The distance from bI to bJ is minimum for all (bI, bJ).

Valid expression Semantics

concatenated

Description

concatenated, given a range whose value_type is a range, concatenates them.

Header

  • <pstade/oven/concatenated.hpp>

Model of

  • Pipable Object

Valid expressions

Valid expression Semantics
_rng|concatenated A Range up to Bidirectional Range {a11,a12,..,a1M1,a21,a22,..,a2M2,..,aN1,aN2,..,aMN}.

Specification

  • SegmentIterator is an iterator of _rng.
  • LocalIterator is an iterator of *_begin(_rng).

Preconditions

  • range_value<_typeof(_rng)>::type is a Range.
  • {a11,a12,..a1MI} is *_begin(_rng), {a21,a22,..a2M2} is *boost::next(_begin(_rng), 1), and so on.
  • A LocalIterator must be valid after copying of SegmentIterator.
Note

The last precondition implies broken_into and xpressive_tokenized can't work with concatenated.

Example 

std::string input("This is his face");
boost::regex re("\\w+");
BOOST_CHECK( equals(
    input|tokenized(re)|concatenated,
    std::string("Thisishisface")
) );

scanned

Description

scanned is similar to std::accumulate, but returns a range of successive reduced values from the base range.

Header

_rng|scanned(s, _fun) A Range up to Forward Range {s,_fun(s, a0),_fun(_fun(s, a0), a1),_fun(_fun(_fun(s, a0), a1), a2),..,aN)}

sliced

Description

sliced provides the column view of the base range.

Note

This name is different from Range Library Proposal, which is the role of offset or window.

Header

  • <pstade/oven/sliced.hpp>

Model of

  • Pipable Object

Valid expressions

Valid expression Semantics
_rndrng|sliced(s, t) {*(f+s+t*1),*(f+s+t*2),*(f+s+t*3),..,*(f+s+t*n)}

Preconditions

  • f = _begin(_rndrng)
  • n = oven::distance(_rndrng)/t
  • n == 0 || n % t == 0 and 0 <= s && s < t.

Example 

int const answer[] = { 2,6,10,14 };
BOOST_CHECK( equals(answer,
    counting(0, 16)|sliced(2, 4)
) );
