transformed

Description

Header

  • <pstade/oven/transformed.hpp>

Model of

  • Range Adaptor

Valid expressions

Valid expression Semantics
op_make_transformed<Ref> A type of Callable Object which can make a transform_iterator range whose reference is Ref from a Function Object.
op_make_transformed<Ref>()(rng, fo) Returns a Range whose iterators behaves as if it were originally wrapped in transform_iterator.
op_make_transformed<> A type of Callable Object which can make a transform_iterator range from a Callable Object.
op_make_transformed<>()(rng, co) Returns a Range whose iterators behaves as if it were originally wrapped in transform_iterator.
make_transformed op_make_transformed<>()
rng|transformed(co) make_transformed(rng, co)

Preconditions

  • fo(*boost::begin(rng)) is a valid expression.
  • co(*boost::begin(rng)) is a valid expression.
