A String Adaptor is a Range Adaptor for strings.

lower_chars

Description

lower_chars performs case-conversion on all characters in the base range.

Header

  • <pstade/oven/lower_chars.hpp>

Model of

  • Pipable Object

Notation

  • loc is a std::locale object.
  • to_l is an imaginary function object to call std::tolower using loc.

Valid expressions

Valid expression Semantics
_rng|lower_chars(loc) _rng|transformed(to_l)
_rng|lower_chars _rng|lower_chars(std::locale())

Preconditions

  • std::tolower(*boost::begin(_rng), loc) is well-formed.

Example 

BOOST_CHECK( equals(
    std::string("AbCdEFG")|lower_chars,
    std::string("abcdefg")
) );

  • to_lower proposal
