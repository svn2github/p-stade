Model of
  • Static Function Object
Valid expressions

Valid expression

Semantics

parallel_copy(s, _fwdrng, it)

copy(_fwdrng, it)

Preconditions
  • _iter_of<_fwdrng>::type および _typeof(it) は Parallel Safe である。

parallel_reduce

Description
Header
  • <pstade/oven/parallel_reduce.hpp>
Model of
  • Static Function Object
Notation
  • V is _value_of<_typeof(_fwdrng)>::type.
Valid expressions

Valid expression

Semantics

parallel_reduce(s, _fwdrng, _fun)

std::accumulate(boost::next(_begin(_fwdrng)), _end(_fwdrng), V(*_begin(_fwdrng)), _fun)

parallel_reduce(s, _fwdrng, z, _fun)

std::accumulate(_begin(_fwdrng), _end(_fwdrng), V(z), _fun)

Preconditions
  • _iter_of<_fwdrng>::type と _fun は Parallel Safe である。
  • s は boost::range_difference<_typeof(_fwdrng)>::type と交換可能なごく小さなサイズである。
  • 最初の式の場合、 !boost::empty(_fwdrng) となる。
  • 対応するセマンティクスは有効な式である。
  • _fun は associative である。
