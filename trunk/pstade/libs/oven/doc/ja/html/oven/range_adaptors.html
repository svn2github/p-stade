Rangeアダプタ(Range Adaptors)は、Rangeに一つ以上の基本的なRangeの変化した表現を与える。 Rangeアダプタは、それらの要素が要求に応じて計算されるという意味でlazyである。 Rangeアダプタは、とくに明記しない限りは以下の要件を満たす：

  • 計算量は O(1); 定数時間で返る。
  • ベースとなるRangeの Range コンセプトを変更しない。

adjacent_filtered

Description

adjacent_filtered は、隣接した組に適用される述語を満たさない間、要素を読み飛ばす。

Header
  • <pstade/oven/adjacent_filtered.hpp>
Model of
  • Pipable Adaptor
Notation
  • {a1,...,aN} は _rng である。
  • {b1,...,bM} は、 {a1,...,aN} の subsequence である。
  • _prd(b1, b2), _prd(b2, b3), ..., _prd(b(M-1), bM) は true である。
  • bI から bJ までの距離は、全ての (bI, bJ) の最小である。
Valid expressions

Valid expression

Semantics

_rng|adjacent_filtered(_prd)

Forward Constant {b1,...,bM}

Preconditions
  • _rng は Readable もしくは Lvalue である。
  • _prd(*_begin(_rng), *_begin(_rng)) は適格である。
  • _prd は Assignable かつ Default Constructible である。
  • _prd は対象を変更しない。
Complexity
  • 対応する _prd を高々 distance(_rng) 回だけ適用する。
Example

applied

Description

applied は、アルゴリズムを表す関数オブジェクトを与えるアダプタを生成する。

Header
  • <pstade/oven/applied.hpp>
Model of
  • Pipable Adaptor
Valid expressions

Valid expression

Semantics

_rng|applied(_fun1, _fun2)

[_fun1(_rng), _fun2(_rng))

_rng|applied(_fun3)

[_begin(r), _end(r)), where r = _fun3(_rng).

Preconditions
  • _typeof(_fun1(_rng)) および _typeof(_fun2(_rng)) は、 _iter_of<_typeof(_rng)>::type に変換可能である。
  • _iter_of<_typeof(fun3(_rng))>::type は _iter_of<_typeof(_rng)>::type に変換可能である。
  • [_fun1(_rng), _fun2(_rng)) は Range でる。
  • r = _fun3(_rng) である [_begin(r), _end(_rng)) は Range である。
Complexity
  • _fun1、 _fun2 および _fun3 に依存する。
Example 
namespace lambda = boost::lambda;
std::string src("abcdefghijk");
std::string s1("efg");
BOOST_CHECK( equals(
    std::string("efghijk"),
    src|applied(lambda::bind(search, lambda::_1, s1), end)
) );
See also
  • applier
  • range_transformer

concatenated

Description

concatenated は、与えられたRangeとRangeを連結する。

Header
  • <pstade/oven/concatenated.hpp>
Model of
  • Pipable Adaptor
Notation
  • SegmentIterator is _iter_of<_typeof(_rng)>::type.
  • LocalRange is _value_of<_typeof(_rng)>::type.
  • LocalIterator is _iter_of<LocalRange>::type.
  • {aI1,aI2,...,aIMI} is *boost::next(_begin(_rng), I).
Valid expressions

Valid expression

Semantics

_rng|concatenated

An up-to-Bidirectional {a11,a12,...,a1M1,a21,a22,...,a2M2,...,aN1,aN2,...,aMN}

Preconditions
  • _rng は Readable もしくは Lvalue である。
  • LocalRange は Range である。
  • SegmentIterator はその破壊によって、以前そのイテレータから得られた LocalIterator を無効にしない。

[重要な最後の事前条件は、 SegmentIterator が LocalIterator オブジェクトの有効期間を管理するのを許さないことを示唆する。 したがって、 broken_into は事前条件を満たすことができない。 (Boost 1.35 の xpressive_tokenized は正しく動作する。) 幸運にも、 memoized はそれらを自明に、 Lvalue Range に変換することができる。]

Example 
std::string input("This is his face");
boost::regex re("\\w+");
BOOST_CHECK( equals(
    input|tokenized(re)|concatenated,
    std::string("Thisishisface")
) );

scanned

Description

scanned is similar to std::accumulate, but returns a range of successive reduced values from the base range.

Header
  • <pstade/oven/scanned.hpp>
Model of
  • Pipable Adaptor
Notation
  • {a1,a2,...,aN} is _rng.
  • {b1,b2,...,bM} is _rng1.
  • S1 is _value_of<_typeof(_rng1)>::type.
Valid expressions

Valid expression

Semantics

_rng|scanned(s, _fun)

An up-to-Forward Readable non-Lvalue Constant {s,_fun(s, a1),_fun(_fun(s, a1), a2),_fun(_fun(_fun(s, a1), a2), a3),...,aN)}

_rng1|scanned(_fun1)

An up-to-Forward Readable non-Lvalue Constant {b1,_fun1(b1, b2),_fun1(_fun(b1, b2), b3),_fun1(_fun1(_fun1(b1, b2), b3), b4),...,bM)}

Note

s is copied, meaning that it is not referenced from Oven later.

Preconditions
  • _rng is Readable or Lvalue.
  • s is Assignable and Copy Constructible.
  • a = _fun(_fun(s, a), a) is well-formed, where _value_of<_typeof(_rng)>::type a = *_begin(_rng)).
  • _rng1 is Readable or Lvalue.
  • S1 is Assignable and Copy Constructible.
  • b = _fun1(_fun1(b, b), b) is well-formed, where _value_of<_typeof(_rng1)>::type b = *_begin(_rng1)).
  • !boost::empty(_rng1)
  • _fun and _fun1 are Assignable and Default Constructible.
Example 
int const src[] = { 1,2,3,4,5 };
std::string null;

BOOST_FOREACH (std::string str, src|scanned(null, &::stringize)) {
    std::cout << "\"" << str << "\" ";
}
// outputs: "" "1" "12" "123" "1234" "12345"

  • scanl
  • scanl1
