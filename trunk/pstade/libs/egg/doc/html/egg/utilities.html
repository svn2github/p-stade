It is difficult or even impossible to get a Function Object from a function template. Egg no longer considers function templates to be useful, so that it provides "objectified" functions which replaces some famous function templates.

lambda_bind

Description

lambda_bind is an Operatable Object which represents boost::lambda::bind.

Header
  • <pstade/egg/lambda/bind.hpp>
Model of
  • Operatable Object
  • Bindable Object
Notation
  • ...
Valid expressions

Valid expression

Semantics

lambda_bind(f, a1,...,aN)

boost::lambda::bind(f, a1,...,aN)

Preconditions
  • 0 <= N && N <= 9.
  • The corresponding semantics is a valid expression.
Example 

// \x -> (\y -> plus(x, y))
BOOST_CHECK( lazy(lambda_bind)(plus, _1, bll::protect(_1))
    (a)(b) == plus(a, b) );

This is currying in Boost.Lambda.

infix

Description

Egg provides FC++ infix operator syntax.

Header
  • <pstade/egg/infix.hpp>
Model of
  • ...
Notation
  • ...
Valid expressions

Valid expression

Semantics

a1 ^_dco^ a2

_dco(_arg_list(a1, by_perfect), _arg_list(a2, by_perfect))

Preconditions
  • using namespace infix; is placed before the valid expression.
  • The corresponding semantics is a valid expression.
Example 
using namespace infix;
BOOST_CHECK( (10 ^plus^ 12 ^plus^ 3) == 25 );
