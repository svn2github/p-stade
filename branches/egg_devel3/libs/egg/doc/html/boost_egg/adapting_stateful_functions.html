Adapting stateful functions

For some reason, you need statically initialized "stateful" functions adapted by Function Adaptors. As you know, the initializer of static_ must be the empty double braces {{}}. How can we pass "states" to Function Adaptors? We need "ugly" macros.

An adapted non-local Function Object with static storage duration is statically initialized if the "base" Function Object type is POD and the expression passed to the macro is a constant expression. Also, Function Adaptors don't modify Default Constructible and Copy Assignable-ness of base Function Object unless otherwise specified.
